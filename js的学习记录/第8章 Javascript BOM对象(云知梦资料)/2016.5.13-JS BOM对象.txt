2016.5.13-BOM对象

BOM全称:browser object model(浏览器对象模型)

浏览器内部对象，地址栏啊，还有文档区（最大的部分）

bom浏览器对象:
• window对象           
• navigator浏览器特性（这个很重要）
• screen屏幕对象   //就四个属性
• history对象（历史区）
• location地址栏（地址栏区）
• document对象（文档区）
------------------------------------------------------------------------------------------------
window对象:
属性:
frames   //窗口数组 （看本页面有多少小窗口）
top      //顶级窗口 （这个就是顶部窗口。）   //可以看到所有的窗口，现在谷歌不行了。就不用了。

方法:
alert();
confirm();
prompt();
setInterval();
clearInterval();
setTimeout();
clearTimeout();
------------------------------------------------------------------------------------------------
Navigator对象:
属性:
userAgent;  //或得浏览器的属性，以及版本
------------------------------------------------------------------------------------------------
screen屏幕对象:
属性:
height
width    //这个就是看自己电脑的分辨率。在做网站的时候会用到。
availHeight //有效的高
availWidth  
------------------------------------------------------------------------------------------------
history对象:（没有属性，只有方法）
方法:
back() 后退一个历史
forward() 前进一个历史
go(-1) 前进一个历史
------------------------------------------------------------------------------------------------
location地址栏:  
属性:
hash       //获取url里的锚点#abc
host       //主机名
href       //当前的url地址  location.href   ,可以获取，可以更改
pathname   //index.html 页面的名称
protocol   //协议 http:  或者是 https:  
search     //参数 ?id=10&name=user1  这个其实就是获取参数部分。

方法:
reload();  //刷新页面
------------------------------------------------------------------------------------------------
